Lógica Cableada

El SISTEMA DE CONTROL DE RELÉS , también conocido como lógica cableada, nos peromite automatizar de manera electromecánica. Un relé de control permite que la corriente eléctrica fluya a través de una bobina conductora que abre o cierra un interruptor. También protege la corriente del circuito. Con un relé de control, los usuarios no necesitan girar manualmente el interruptor para aislar o cambiar el estado de un circuito eléctrico.

CADESIMU: Diagramas de Bloques

El diagrama de bloques de función FBD, también conocido como DIAGRAMA DE BLOQUES o BDF, es un lenguaje gráfico para el diseño de controladores lógicos programables, que puede describir la función entre las variables de entrada y las variables de salida. Una función se describe como un conjunto de bloques elementales. Las variables de entrada y salida se conectan a los bloques mediante líneas de conexión.

CADESIMU: Diagrama de Escalera

El lenguaje ladder, también conocido como DIAGRAMA DE ESCALERA es un lenguaje de programación gráfico que se utiliza para desarrollar software para controladores lógicos programables. Es uno de los lenguajes que la norma IEC 61131 especifica para su uso con los PLC. Un programa en notación de diagrama de escalera es un diagrama de circuito que emula los circuitos del hardware lógico de los relés.
